(PROJECT OVERVIEW)

A Workflow for Sanity.

HexTodo is a Practical SOP Manager built with love. It integrates a Task List with an Asset/Knowledge-Base engine to overcome Information Overload.

"Focus on operation, not on managing." No duplication—preview files exactly where they sit on your computer.

(OBJECTIVES)
  • 01

    Step Process: Breaking big goals into serious, manageable steps.

  • 02

    SOP Guided Task: Assigning tasks without repeating instructions (Shared SOPs).

(HISTORIES & PHILOSOPHIES)

Built on Universal Truths.

The Timeline

"From Egyptian papyrus lists (2400 BCE) to modern digital checklists..."

Digital To-Do origins in the 80s digitized portability. HexTodo returns to the Hybrid Approach: Waterfall planning (Long-term) with Agile execution (Daily).

The Formula

Task = Time [Clock & Calendar] + Resources [Assets, People, Knowledge-Based]

Our goal: Finish the list—not follow the order. Match tasks to the energy you can afford.

Functional Architecture

The Hex Engine Components.

(Home Dashboard)

User Greet • Quote of the Day • Counts for Todo/Doing/Done/Overdue • Category Summary • Smart Calendar with colors [G/R/Y/B].

(Task Panel)

[Now] Focus Pin • [Today] 3-7 Task Limit • [Upcoming] Horizon • Priority Temperature Sliders • Manual Sorting.

(Task)

(SOP Checklist) with Step Processes • (Media Systems) Internal Player for local files/URLs • Recurrence rules & Priority levels.

(People)

Workload viewing • Names & Roles • Assigning tasks to [Role, Team, Group] • Profile Avatars.

(Assets)

Bookmark Manager with Webpage Snapshots • Reusable Resources • Search-to-Attach logic for SOP steps.

(Knowledge-Based)

Markdown Engine for instructions • Import/Export [Txt, Md, Docx, Pdf] • Overcoming info-overload via technical reference guides.

(Quick Settings)

Light/Dark Toggle • (Category) Manager • (Archive Decision Point): Delete, Reuse (keep SOP), or Restore.

(Space/Workplace)

Isolated environments for Personal/Work/Hobby • Unique SOPs, Assets, and People per workspace.

(Settings)

(User Info) & Country data • (Themes Style) • (Task Behaviours) for archive rules and focus limits.

(UX & UI SYSTEM)

Healthy Experience Architecture.

"Real things to destroy toxic feelings..."

Strikethrough Gestures for deletion. Bottom-aligned [Add Task] triggers to build proactive habits. Expandable layers for focus freedom.

Healthy Defaults

  • • Reduced Cognitive Load
  • • Full Keyboard Accessibility
  • • [Now] Maximum limit (1-3)
  • • Relative Time Formatting
